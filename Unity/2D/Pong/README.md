## Description
Well...it's Pong.

## Features
- 2 player game
- Score keeper

## Controls

| Left Paddle |
| ----------- |
| Up = `w`    |
| Down = `s`  |

| Right Paddle            |
| ----------------------- |
| Up = `up arrow key`     |
| Down = `down arrow key` |
